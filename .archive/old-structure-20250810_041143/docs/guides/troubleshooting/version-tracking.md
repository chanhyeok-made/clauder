# 버전 추적 문제

## "버전 메타데이터 없음"

### 증상
```
⚠️ README.md - 버전 메타데이터 없음
```

### 해결
```
/clauder track add README.md
```

### 수동 추가
```yaml
---
doc_id: [ID]
---
```

## "순환 참조 감지"

### 원인
A → B → C → A 형태의 참조

### 해결

1. **순환 참조 확인**
   ```
   /clauder track check all
   ```

2. **의존성 재검토**
   - 불필요한 참조 제거
   - 단방향 참조로 수정

3. **구조 개선**
   - 계층적 구조로 변경
   - 공통 모듈 분리

## doc_id 관리

### 중복 ID 확인
```bash
/clauder track check --duplicates
```

### ID 할당 규칙
- 1-99: 핵심 원칙
- 100-199: 명령어
- 200-299: 설계 문서
- 300-399: 가이드
- 400-499: 템플릿
- 500-599: 기타
- 700+: 프로젝트 문서
