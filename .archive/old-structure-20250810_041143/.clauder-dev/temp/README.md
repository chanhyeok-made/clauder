# 임시 작업 공간

> 개발 중 임시로 사용하는 파일들을 위한 공간입니다.

## 📁 구조

- **experiments/**: 실험적 코드
- **drafts/**: 초안 문서
- **scratch/**: 스크래치 파일

## 🔍 주요 내용

정식 문서나 코드로 만들기 전 아이디어를 테스트하고 실험하는 공간입니다. 이 디렉토리의 내용은 언제든 삭제될 수 있습니다.

## ⚠️ 주의사항

- Git에 커밋하지 않음 (.gitignore에 포함)
- 중요한 내용은 정식 위치로 이동
- 주기적으로 정리
- 다른 파일에서 참조하지 않음

## 🧹 정리 정책

- 30일 이상 수정되지 않은 파일 삭제
- 완료된 실험은 learnings로 이동
- 채택된 아이디어는 정식 문서화
