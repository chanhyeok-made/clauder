# 일일 작업 로그

> 매일의 개발 작업을 기록하는 공간입니다.

## 📁 구조

- **YYYY-MM-DD.md**: 날짜별 로그 파일
- **weekly-summary.md**: 주간 요약
- **in-progress.md**: 진행 중인 작업

## 🔍 주요 내용

매일의 작업 내용, 해결한 이슈, 새로운 발견, 다음 작업 계획 등을 상세히 기록합니다.

## ✍️ 작성 가이드

### 일일 로그 템플릿
```markdown
# YYYY-MM-DD

## 완료한 작업
- [ ] 작업 1
- [ ] 작업 2

## 발견한 이슈
- 이슈 설명
- 해결 방법

## 내일 계획
- 작업 계획

## 메모
- 기타 특이사항
```

## 📌 작성 시점

- 작업 시작 시: 계획 작성
- 작업 중: 이슈와 해결 기록
- 작업 종료 시: 완료 사항 체크
