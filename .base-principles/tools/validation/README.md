# 검증 도구 템플릿

> 프로젝트별 검증 도구를 만들기 위한 템플릿입니다.

## 📁 구조

- **@level1-check.sh**: Level 1 경량 검증 템플릿
- **@level2-check.sh**: Level 2 표준 검증 템플릿

## 🔍 사용 방법

1. 템플릿을 프로젝트에 복사
2. 프로젝트별 검증 로직 추가
3. 실행 권한 부여: `chmod +x script.sh`

## 📌 커스터마이징 포인트

- 파일 형식별 검증
- 프로젝트별 규칙
- 커밋 메시지 형식
- 테스트 명령어
