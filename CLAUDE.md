# Claude Code 프로젝트 가이드

> 💡 이 프로젝트는 아직 초기화되지 않았습니다.
> 
> 초기화하려면 다음 명령을 실행하세요:
> ```
> /clauder initialize
> ```
> 
> 사용 가능한 명령어:
> - `/clauder initialize` - 프로젝트 초기화
> - `/clauder check` - 설치 상태 확인
> - `/clauder generate` - 문서 생성
> 
> 자세한 정보: @.claude/README.md
