모든 작업은 다음 단계를 따릅니다:

1. **분석** → @.claude/workflow/01-analysis.md
2. **작업** → @.claude/workflow/02-implementation.md  
3. **회고** → @.claude/workflow/03-retrospective.md
4. **문서화** → @.claude/workflow/04-documentation.md
5. **커밋** → @.claude/workflow/05-commit.md
