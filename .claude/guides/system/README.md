# 시스템 가이드

> Claude Code의 시스템 레벨 동작과 자동화를 위한 가이드입니다.

## 📁 구조

- **hooks.md**: 자동화 훅 동작 정의

## 🔍 주요 내용

Claude Code가 자동으로 수행해야 하는 시스템 레벨 동작들을 정의합니다. 파일 편집 시 자동 검증, 참조 확인, 버전 동기화 등의 훅 동작을 다룹니다.

## 📌 자동화 원칙

1. 사용자 개입 없이 자동 실행
2. 투명하고 예측 가능한 동작
3. 오류 시 명확한 피드백
