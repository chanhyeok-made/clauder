# /clauder add

프로젝트 요소 추가

## 고유 기능
```
add context <name>    # 새 컨텍스트
add override <file>   # 템플릿 오버라이드
add alias <name>      # 경로 별칭
```

## 예시
```
/clauder add context debugging
→ .claude/custom/contexts/debugging.md 생성
```
