## 명령어 레퍼런스

모든 Clauder 명령어에 대한 상세 설명은 [명령어 문서](commands/)를 참조하세요.

### 주요 명령어
- `/clauder start` - 프로젝트 초기화
- `/clauder daily` - 일일 상태 점검
- `/clauder check` - 문서 상태 확인
